# Assets do MoneyGen

## Ícones necessários:

### icon.png
- **Tamanho**: 1024x1024 pixels
- **Formato**: PNG com fundo transparente ou sólido
- **Uso**: Ícone principal do app

### adaptive-icon.png  
- **Tamanho**: 1024x1024 pixels
- **Formato**: PNG com fundo transparente
- **Uso**: Ícone adaptativo Android (apenas o foreground)

### splash.png
- **Tamanho**: 1284x2778 pixels (ou similar)
- **Formato**: PNG
- **Uso**: Tela de carregamento

## Como criar:

1. **Design**: Crie um ícone com tema financeiro (cifrão, gráfico, moeda)
2. **Cores**: Use o tema do app (roxo #8b5cf6, preto #000)
3. **Ferramentas**: Canva, Figma, ou qualquer editor gráfico
4. **Substitua**: Coloque os arquivos nesta pasta com os nomes exatos
